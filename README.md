# 📘 교육과정 편성 자율점검 (v5)

2022 개정 교육과정 기준으로 고등학교 학점 배당표(.xlsx)를 21개 항목으로 자동 점검합니다.

## ✨ 주요 기능

- **다중 파일 업로드**: 여러 학교 엑셀을 동시에 점검
- **21개 점검 항목 자동 실행** (A 학점 총량, B 학기 배분, C 과목별 학점, D 교과군별, E 위계, F 형식, G 정성)
- **엑셀 다운로드**: 요약 시트 + 전체결과 시트 + 학교별 시트 + 비고 시트
- **2022 개정 정식 과목명 검증** (145개 과목 DB 내장)
- **위계 과목 선후 순서 검증** (12쌍)
- **다중 시트 자동 선택** (교육과정 시트 자동 탐지)

## 🚀 빠른 시작

### 로컬 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

### Streamlit Cloud 배포
1. 이 폴더 전체를 GitHub Public 저장소에 push
2. https://share.streamlit.io 접속 → New app
3. Repository / Branch / Main file path(`app.py`) 지정 → Deploy

## 📊 점검 항목 (21개)

| 코드 | 항목 |
|------|------|
| A1~A5 | 총 이수학점 / 교과 / 필수 / 창체 / 초과 |
| B1~B3 | 학기 균형 / 학기 단위 이수 / 창체 편중 |
| C1~C3 | 학점 증감 범위 / 한국사 / 동일과목 동일학점 |
| D1~D2 | 국·수·영 81학점 / 체육 10학점 |
| E1~E2 | 공통→선택 / 위계 순서 |
| F1~F3 | 과목명 / 형식 / 기록 |
| G1~G3 | 선택권 / 공통 전원 / 종교 |

## 💾 엑셀 다운로드 구조

다운로드 파일은 다음 시트를 포함합니다:
- `요약`: 학교별 PASS/FAIL/CHECK 카운트
- `전체결과`: 모든 학교 전체 점검 결과 통합
- `<학교명>`: 학교별 상세 점검 결과
- `<학교명>_비고`: 학교별 비고/유의사항

## 🛠 트러블슈팅

**`헤더 행을 찾지 못했습니다`**
- 시트가 여러 개일 경우 자동 선택되지만, 헤더에 `구분/교과/과목/학점/학년` 키워드가 너무 적으면 실패할 수 있습니다.
- 학점 배당표 시트의 상단 헤더에 위 키워드가 포함되어 있는지 확인하세요.

**`Styler.applymap is deprecated`**
- pandas 2.1+ 환경에서 `Styler.map`을 자동 사용합니다. requirements.txt에 `pandas>=2.1.0` 명시되어 있습니다.
